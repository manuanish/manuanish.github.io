## Piglin Barter Probabilities
In this section I calculte the odds of receiving $12$ ender pearls by trading $234$ gold ingots.
<br/>
#### Traded Gold
In the run, I trade a total of $26$ gold blocks, which can be translated to $26 \cdot 9 = \boxed{234}$ gold ingots in total.
<br/>
### Odds of rolling a pearl trade
First, we unzip the `.minecraft\versions\1.19\1.19.jar` file and navigate the data folder to find the piglin bartering loot tables for Minecraft 1.19 : `1.19.jar\data\minecraft\loot_tables\gameplay\piglin_bartering.json`.
This is given as follows: (list is truncated to a small amount, you can view the entire file on [GitHub](https://github.com/manuanish/minecraft-119-record/blob/main/piglin_bartering.json))
<br/>
```json
{
  "type": "minecraft:barter",
  "pools": [
    {
      "bonus_rolls": 0.0,
      "entries": [ ... ],
      "rolls": 1.0
    }
  ]
}
```
<br/>
#### Calculating sum of weights
As we can see, each item in the `"entries"` data point has a given `"weight"` (which we will call $w$), and using this we can calculate the probability of
receiving any item $i$ in the follow way: (we denote $w_i$ to be the `"weight"` of any item i)
<br/>
$$
P(i) = \dfrac{w_i}{\displaystyle \sum_{\forall \ f \ \in \ \texttt{piglin\_bartering.json} }{w_f}}
$$
<br/>
We know that:
$$
W = \sum_{\forall \ f \ \in \ \texttt{piglin\_bartering.json} }{w_f}
$$
$W$ is going to be constant, so we can find this by adding all the weights in the `piglin_bartering.json` file. Doing this we get that:
$$
W = 459
$$
<br/>
#### Final pearl trade probability
If we inspect the `piglin_barters.json` file we can see that the `"weight"` of a pearl trade is $10$
```json
{
  "type": "minecraft:item",
  "functions": [
    {
      "add": false,
      "count": {
        "type": "minecraft:uniform",
        "max": 4.0,
        "min": 2.0
      },
      "function": "minecraft:set_count"
    }
  ],
  "name": "minecraft:ender_pearl",
  "weight": 10
},
```
Therefore, the probability of receiving a pearl trade from $1$ gold ingot is given by:
$$
\boxed{\frac{10}{459}} \approx 0.022
$$
### Calculating my probability
A single pearl trade can give you a maximum of $4$ pearls, or a minimum of $2$ pearls (as shown above). In my run, I recieved exactly $12$ ender pearls
so I may have recieved any where in between $3$ pearl trades or $6$ pearl trades.
<br/>
For the sake of simplicity, I will calculate the probability at each of $3$ and $6$ pearl trades and average it to find the general likelyhood.
<br/>
#### Calculating probability for $n$ pearl trades (general case)
From the above, we may ask the question:
> Given that the probability of rolling $1$ pearl trade from $1$ gold ingot is $\tfrac{10}{459}$,
> what is the probability that a runner recieves $n$ pearl trades from $g$ gold ingots?

<br/>
We can answer this using the binomial theorem.
Begin by noting that $n \leq g$ by definition (if otherwise, then the runner is receiving multiple trades from a single ingot).
Therefore, the number of ways that one can choose the $n$ pearl trades from the total $g$ gold nuggets traded is:
$$
{g \choose n} = \frac{g!}{n!(g-n)!}
$$
The probability of $n$ pearl trades is simply given by:
$$
\left(\frac{10}{459}\right)^n \cdot \left(\frac{449}{459}\right)^{g-n}
$$
And so the final probability is given as shown below:
$$
P = \boxed{\frac{g!}{n!(g-n)!}\cdot\left(\frac{10}{459}\right)^n \cdot \left(\frac{449}{459}\right)^{g-n}}
$$
<br/>
#### Case 1: $3$ pearl trades
Using the equation above, we can calculte the probability of getting $3$ pearl trades from $234$ gold ingots. This is simply:
$$
P_1 = \frac{234!}{3!(231)!}\cdot\left(\frac{10}{459}\right)^3 \cdot \left(\frac{449}{459}\right)^{231} \approx \boxed{0.1344}
$$
<br/>
#### Case 2: $6$ pearl trades
Using the equation above, we can calculte the probability of getting $3$ pearl trades from $234$ gold ingots. This is simply:
$$
P_1 = \frac{234!}{6!(228)!}\cdot\left(\frac{10}{459}\right)^6 \cdot \left(\frac{449}{459}\right)^{228} \approx \boxed{0.1506}
$$
<br/>
Averaging the both of these, we get the probability of this event is
$$
\frac{0.1344 + 0.1506}{2} \approx \boxed{\boxed{0.14255}}
$$
<br/>
<br/>
### Was this unlucky or lucky?
The expected number of pearl trades from $234$ gold ingots is given by
$$
\frac{10}{459} \cdot 234 \approx 5.098
$$
and the number of pearl trades I recieved (on average) is:
$$
\frac{3 + 6}{2} = 4.5 \quad \implies 4.5 < 5.098
$$
Thus, we can conclude that this run was *slightly unlucky* due to the fact that I recieved less pearl trades than the expected outcome.
