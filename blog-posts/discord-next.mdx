## Tutorial
Before we can begin, we need to create a discord [webhook](https://en.wikipedia.org/wiki/Webhook) for our desired server.
### <Twemoji emoji="ðŸ”—"/> Create a new webhook
To create a new webhook, we first have to navigate to **Server settings > Integrations**
<br/>
<BlogImage src="/blog/discord-next/integrations-dropdown.png" width="1619" height="1080" />
<br/>
<BlogImage src="/blog/discord-next/integrations-page.png" width="1920" height="1080" />
<br/>
Now we can click **Create Webhook**
<br/>
<BlogImage src="/blog/discord-next/new-webhook.png" width="1920" height="1080" />
<br/>
Finally we can click on **Copy Webhook URL**
<br/>
<BlogImage src="/blog/discord-next/copy-webhook-url.png" width="1920" height="1080" />
<br/>
### <Twemoji emoji="ðŸš€"/> Sending messages
The URL is of the form:
```
https://discord.com/api/webhooks/[webhook_id]/[webhook_token]
```
The API endpoint expects a POST response in the following manner:
```js
fetch("https://discord.com/api/webhooks/[webhook_id]/[webhook_token]", {
  method: "post",
  headers: {
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    content: "Hello World!",  // Message Content
    username: username,  // Webhook Username
  }),
});
```
